# tokens

## 是什么？

tokens 是一个可以申请/释放随机码的随机池工具。

## 会什么？

1. 随机获取口令
2. 释放已经申请的口令
3. 补充新的可用口令（填充口令池

### 其他

1. 提供方法以获取运行状态
    
    - 当前总口令数
    - 已用口令数
    - 剩余可用口令数
    - 最大可增长空间
    
    持续补充...
    
2. 提供口令池备份、恢复方法和接口

## 优点

1. **快速完成**口令的申请/释放
    
    使用口令池，初始化时即生成大量口令
    
    避免在"申请口令"操作中，耦合口令生成逻辑和冲突处理逻辑
    
    （降低申请操作的执行速度；增加了代码维护成本）
    
2. 运行状态可控
    提供接口，可以获取丰富的运行状态
    
3. 高可用
    提供口令池备份、恢复方法；也可以基于接口，快速实现自定义的备份、恢复操作

### 使用场景举例

1. OAuth
2. short url
3. 验证码
4. 临时口令

other...
